# 👑 OVERLORD

**OVERLORD** is a high-performance, terminal-based stream controller designed for power users who demand efficiency and control. Built with Go, it provides a seamless interface to search for movies and TV series and launch them directly in your browser.

## 🚀 Features
- **Fast & Lightweight:** Written in Go for optimal performance.
- **CLI-First Workflow:** No clutter, just search and stream.
- **Dynamic Selection:** Built-in support for series (Season/Episode selection).
- **Cross-Platform:** Designed to work natively on Linux and Windows.
- **Clean Interface:** Professional UI powered by `survey`.
